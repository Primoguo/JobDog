# Markdown 样式集合

## 一、文本样式

| 样式名称 | Markdown 语法 | 渲染效果 | 说明 |
|:---------|:--------------|:---------|:-----|
| 加粗 | `**文本**` 或 `__文本__` | **加粗文本** | 标准语法 |
| 斜体 | `*文本*` 或 `_文本_` | *斜体文本* | 标准语法 |
| 加粗斜体 | `***文本***` | ***加粗斜体*** | 标准语法 |
| 删除线 | `~~文本~~` | ~~删除线~~ | GFM扩展 |
| 高亮 | `==文本==` | ==高亮文本== | 扩展语法 |
| 下标 | `H~2~O` | H~2~O | 扩展语法 |
| 上标 | `X^2^` | X^2^ | 扩展语法 |
| 行内代码 | `` `代码` `` | `code` | 标准语法 |
| 下划线 | `<u>文本</u>` | <u>下划线</u> | HTML标签 |
| 键盘按键 | `<kbd>Ctrl</kbd>` | <kbd>Ctrl</kbd> | HTML标签 |
| 标记 | `<mark>文本</mark>` | <mark>标记</mark> | HTML标签 |
| 小号字体 | `<small>文本</small>` | <small>小号</small> | HTML标签 |
| 大号字体 | `<big>文本</big>` | <big>大号</big> | HTML标签 |

## 二、标题样式

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 一级标题 | `# 标题` | 最大 |
| 二级标题 | `## 标题` | |
| 三级标题 | `### 标题` | |
| 四级标题 | `#### 标题` | |
| 五级标题 | `##### 标题` | |
| 六级标题 | `###### 标题` | 最小 |
| 一级标题(备选) | `标题` + 下一行 `===` | Setext风格 |
| 二级标题(备选) | `标题` + 下一行 `---` | Setext风格 |

---

## 三、列表样式

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 无序列表 | `- 项目` 或 `* 项目` 或 `+ 项目` | 可混用 |
| 有序列表 | `1. 项目` | 数字+点 |
| 嵌套列表 | 缩进2-4空格 | 可多级嵌套 |
| 任务列表(未完成) | `- [ ] 任务` | GFM扩展 |
| 任务列表(已完成) | `- [x] 任务` | GFM扩展 |
| 定义列表 | `术语` + 换行 + `: 定义` | 扩展语法 |

---

## 四、引用与代码

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 引用块 | `> 引用内容` | 可多行 |
| 嵌套引用 | `>> 二级引用` | 可多级 |
| 行内代码 | `` `code` `` | 单反引号 |
| 代码块(缩进) | 4空格或1Tab缩进 | 标准语法 |
| 代码块(围栏) | ` ```language ` + 代码 + ` ``` ` | GFM扩展 |
| 代码块(语法高亮) | ` ```python ` | 指定语言 |

## 五、链接与图片

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 内联链接 | `[文本](URL)` | 标准语法 |
| 带标题链接 | `[文本](URL "标题")` | 悬停显示标题 |
| 引用链接 | `[文本][id]` + `[id]: URL` | 便于复用 |
| 自动链接 | `<https://example.com>` | 尖括号包裹 |
| 邮箱链接 | `<email@example.com>` | 自动转mailto |
| 锚点链接 | `[文本](#标题)` | 页内跳转 |
| 基础图片 | `![alt](URL)` | 标准语法 |
| 带标题图片 | `![alt](URL "标题")` | 悬停显示标题 |
| 引用图片 | `![alt][id]` + `[id]: URL` | 便于复用 |
| 图片链接 | `[![alt](图片URL)](链接URL)` | 可点击图片 |
| 图片尺寸 | `<img src="URL" width="200">` | HTML标签 |

## 六、表格样式

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 基础表格 | `\| 列1 \| 列2 \|` | GFM扩展 |
| 左对齐 | `:---` | 冒号在左 |
| 居中对齐 | `:---:` | 冒号两边 |
| 右对齐 | `---:` | 冒号在右 |
| 单元格换行 | `内容<br>换行` | HTML标签 |
| 合并单元格 | `<td colspan="2">` | HTML标签 |

## 七、分割与排版

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 分割线 | `---` 或 `***` 或 `___` | 3个及以上 |
| 换行 | 行尾2个空格 或 `<br>` | |
| 段落 | 空一行 | |
| 折叠块 | `<details><summary>标题</summary>内容</details>` | HTML标签 |
| 居中 | `<center>内容</center>` | HTML标签 |
| 右对齐 | `<div align="right">内容</div>` | HTML标签 |

---

## 八、扩展语法

| 样式名称 | Markdown 语法 | 说明 |
|:---------|:--------------|:-----|
| 脚注 | `文本[^1]` + `[^1]: 注释内容` | 扩展语法 |
| 行内数学公式 | `$E=mc^2$` | LaTeX |
| 块级数学公式 | `$$公式$$` | LaTeX |
| 缩写定义 | `*[HTML]: HyperText Markup Language` | 扩展语法 |
| Emoji | `:smile:` `:+1:` | 表情符号 |
| 目录生成 | `[TOC]` | 扩展语法 |
| Mermaid图 | ` ```mermaid ` | 流程图等 |
| 注释 | `<!-- 注释 -->` | 不显示 |
